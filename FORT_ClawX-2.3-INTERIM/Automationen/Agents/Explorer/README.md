# Explorer Agent - Informationsfinder & Echtheitsprüfer mit 5-Quellen-Pflicht 🧭

## Kurzübersicht
**Explorer** ist ein Funktions-Agent, der Informationen aus verschiedenen Quellen findet, aufbereitet und auf ihre Echtheit überprüft.

## Hauptfunktionen
- 🔍 **Informations-Suche:** Findet relevante Daten aus Web, Dateien und Gedächtnis
- 📦 **Aufbereitung:** Strukturiert und fasst Informationen zusammen
- ✅ **Echtheitsprüfung:** Validiert Quellen nach dem Multi-Source Framework

## 5-Quellen-Pflicht
Jede Recherche basiert auf mindestens 5 unabhängigen Quellen für maximale Zuverlässigkeit.

## Einsatzgebiete
- Recherche zu aktuellen Themen
- Faktenüberprüfung von Behauptungen
- Informations-Synthese aus verschiedenen Quellen
- Bias-Erkennung in Berichten

## Beispiel-Aufruf
```bash
# Explorer wird über OpenClaw aktiviert und kann direkt angefragt werden:
"Explorer, finde heraus ob [Thema] wahr ist."
```
