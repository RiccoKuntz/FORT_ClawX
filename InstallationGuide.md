# Installationsguide

## DE
### Installation
1. Installiere openclaw (https://openclaw.ai/)
2. Downloade FORT_ClawX in der Version deiner Wahl (tar.gz).
3. Entpacke FORT_ClawX.tar.gz-Archiv
4. Ersetze deinen ~/.openclaw/workspace Ordner mit dem aus dem Archiv

### Installationshinweise
 - FORT_ClawX bis version 2.0 hat noch keinen Autoinstaller. Du musst also Garvis bitten einen scan durchzuführen und die Pfade auf dein System anzupassen
 - FORT_ClawX ist noch eine Alpha. Sei also vorsichtig.
 - Einsatz nur unter Laborbedingungen und nie in Produktivsystemen. (Linux Mint VM)
 - Viel Spaß mit FORT_ClawX

## EN (AI TRANSLATE)
### Installation
1. Install OpenClaw (https://openclaw.ai/)
2. Download FORT_ClawX in the version of your choice (tar.gz).
3. Extract the FORT_ClawX.tar.gz archive.
4. Replace your ~/.openclaw/workspace folder with the one from the archive.

### Installation Notes
- Up to version 2.0, FORT_ClawX does not feature an automatic installer. You must instruct Garvis to run a scan and adapt the paths to your system.
- FORT_ClawX is still in Alpha. Please handle with care.
- For laboratory and testing environments only; never deploy on production systems. (Linux Mint VM)
- Have fun with FORT_ClawX!
